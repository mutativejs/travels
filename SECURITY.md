# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release of `travels`.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository. Do not file public issues for vulnerabilities.

Include:

- affected version
- reproduction steps
- impact and affected APIs
- any known workaround

Maintainers will acknowledge valid reports as soon as practical, triage severity, and publish a patched release when needed.
